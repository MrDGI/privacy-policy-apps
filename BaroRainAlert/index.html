Política de Privacidad – BaroRainAlert

Última actualización: 2025

1. Información general

BaroRainAlert es una aplicación para Wear OS que analiza los cambios de presión atmosférica del dispositivo para estimar la probabilidad de lluvia. La aplicación está diseñada para funcionar completamente en el dispositivo y no recopila datos personales del usuario.

2. Datos que recopila la aplicación

BaroRainAlert utiliza exclusivamente el sensor barométrico del reloj para obtener valores de presión atmosférica. Estos datos:

  • No se almacenan fuera del dispositivo.
  • No se envían a servidores externos.
  • No se comparten con terceros.
  • No se asocian a ninguna identidad personal.

3. Permisos utilizados

La aplicación puede requerir acceso al sensor barométrico del dispositivo. Este permiso es necesario para su funcionamiento básico y no implica acceso a datos personales.

4. Uso de los datos

Los valores de presión atmosférica se utilizan únicamente para:

  • Calcular tendencias barométricas.
  • Generar alertas locales de posible lluvia.

Toda la información se procesa de forma local en el reloj.

5. Servicios de terceros

BaroRainAlert no utiliza servicios de terceros, no integra publicidad y no emplea herramientas de análisis.

6. Seguridad

Dado que la aplicación no recopila ni transmite datos personales, el riesgo de exposición de información es mínimo. Aun así, se siguen las prácticas recomendadas de desarrollo seguro para Wear OS.

7. Cambios en esta política

Esta política de privacidad puede actualizarse ocasionalmente. Las modificaciones se publicarán en esta misma página.

8. Contacto

Si tienes preguntas sobre esta política de privacidad, puedes contactar con el desarrollador a través de la página de Google Play de la aplicación.
